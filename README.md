# 💻 Hacettepe University - BBM 203: Data Structures Laboratory

This repository contains advanced C++ projects developed during the **BBM 203** course at **Hacettepe University** (Fall 2025). These projects focus on the implementation of fundamental and complex data structures to solve real-world engineering problems.

## 🚀 Projects Overview

| Project | Description | Data Structures Used |
| :--- | :--- | :--- |
| **[Rail Marshal](./BBM203-PA1)** | A train classification and logistics simulation. | Doubly Linked Lists, Stacks |
| **[QuakeAssist](./BBM203-PA2)** | A disaster relief coordination and mission tracking system. | Queues (FIFO), Stacks (LIFO) |
| **[Artifact Stewardship](./BBM203-PA3)** | A high-performance artifact cataloging and assignment system. | AVL Trees, Red-Black Trees |

## 🛠 Technical Competencies
* **Memory Management:** Extensive experience with manual memory allocation, pointers, and preventing memory leaks using RAII principles in C++.
* **Advanced Algorithmic Logic:** * Implementation of **Self-Balancing Trees** (AVL & Red-Black) to ensure $O(\log n)$ search efficiency.
    * Coordination of multiple data structures (e.g., combining Queues for requests and Stacks for missions).
* **System Architecture:** Developing modular, object-oriented software with a clear separation of concerns using header (.h) and implementation (.cpp) files.
* **Performance Optimization:** Using appropriate data structures to minimize time complexity in logistics and search operations.

## 🔧 Tools & Environment
* **Language:** C++11 / C++14
* **Compiler:** GCC / G++
* **Standards:** Clean Code, Professional Documentation, and Robust Error Handling.
